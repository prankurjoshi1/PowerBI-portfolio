# PowerBI-portfolio

Business Intelligence dashboards built using **Power BI Desktop**, demonstrating KPI design, trend analysis, interactive reporting, and data-driven decision support using sample and public datasets.

The focus of these projects is on analytical clarity, clean data modeling, and business usability rather than heavy visual styling.

---

## Tools & Skills

- Power BI Desktop
- Data modeling and measures (DAX)
- KPI definition and reporting
- Trend and segmentation analysis
- Interactive dashboards with slicers
- Business-focused data storytelling

---

## Dashboards

### 1. Vehicle Sales Performance Dashboard

This dashboard analyzes vehicle sales performance using structured transactional data, with the objective of understanding sales trends and product performance.

**Key features:**
- Total sales and revenue KPIs
- Sales trends over time
- Breakdown by product line and deal size
- Interactive filtering for focused analysis

The dashboard is designed for sales and management stakeholders to quickly assess performance and identify key drivers.

### 2. HR Analytics: Employee Attrition Dashboard

This dashboard explores employee attrition patterns using HR data, focusing on identifying workforce segments with higher attrition risk.

**Key features:**
- Overall attrition rate and total employee count
- Attrition by department
- Attrition by age group using custom age bands
- Interactive slicers for exploratory HR analysis

The dashboard emphasizes analytical structure and decision support for HR and management teams.
